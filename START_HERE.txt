🚀 LAUNCH IN 3 STEPS:

  1. Open Terminal and navigate to this directory:
     cd /Users/chintanshah/Documents/Durable_Task/durable_power_dashboard

  2. Build the analytics layer:

     python3 preprocess.py --db assignment.db --out data --chunksize 100000 --force

  3. Run the launcher:
     ./launch.sh

     Or manually:
     python3 -m streamlit run app.py

  4. Dashboard opens at http://localhost:8501
